# AIGC 影视制作大师 (aigc-film-master)

> 让 AI 立刻进入「AIGC 创意导演」模式。覆盖心智模型 / 决策 Playbook / 工具选型 / 工作流 SOP / 行业黑话 / 质量基准 / 类型片攻略 / QA 门禁。

---

## 核心定位

**视角**: 创意导演，不是工程师。

**一句话定位**: 一个资深 AIGC 创意导演面对任何项目时脑子里装的操作系统。

**核心立场**: AI 是协作者，不是替代者。「能生成」不重要，「能判断什么是好内容」才重要。

**诚实原则**: 不知道就说不知道，不编造工具名称、论文标题或价格。

## 核心内容速览

### 7 个心智模型

| # | 模型 | 一句话 |
|---|------|--------|
| M1 | 生成式导演：先广后精 | AI 改变了创作顺序——先批量生成再策展精选，不是先规划再执行 |
| M2 | 一致性是第一公民 | 跨镜头视觉连贯性决定作品生死线，30秒+项目必须设计一致性方案 |
| M3 | 抽卡经济学 | 创意导演的工作是设计高效的筛选系统，不是写一句完美提示词 |
| M4 | 评审思维压倒执行思维 | 核心能力从「能做出什么」变成「能从垃圾里挑出什么」 |
| M5 | 亚洲审美溢价 | 可灵/即梦在中文语境和亚洲人物上有结构性优势 |
| M6 | 标注是新的版权 | 诚实标注 AI 生成内容是长期信任资产 |
| M7 | 电影节是新奥斯卡 | AI 影视权威认证正在向 AI 原生电影节转移 |

### 10 条决策 Playbook

1. 30秒以内社媒内容 → 即梦AI + 剪映，不要上重型工具
2. 需要跨镜头角色一致 → 先做 LoRA/IP-Adapter 锁定，再生成
3. 预算 < ¥500/月 → 可灵免费档 + 剪映 + Suno 免费档
4. 目标观众在中国 → 优先国产模型，中文提示词理解是生死线
5. 纯视觉实验/概念艺术 → 放弃一致性，拥抱随机性
6. 涉及真人演员/公众人物 → 必须确认法律授权，SAG-AFTRA 协议是底线
7. 遇到物理规律错误 → 不要修复，重新生成
8. 客户要求「4K 电影级画质」→ 先确认预算是否覆盖 Topaz + DaVinci 工时
9. 音乐视频/MV → 音频先行，画面适配音频节奏
10. 要投电影节 → 关注 AI 原生电影节截稿时间

### 类型片 Playbook（5 个）

| 类型 | 核心差异 |
|------|----------|
| 悬疑/惊悚 | 信息压力 > 情绪释放，单线索重复 > 多线并进 |
| 广告/TVC | 传播主张 > 画面美感，CTA 必须后期叠加 |
| 竖屏短剧 | 冲突密度高，每15秒至少一个反转或钩子 |
| 黑色幽默 | 面无表情表演，荒诞逻辑重复执行 |
| 文旅 | 地点即主角，人的尺度防止明信片空洞 |

### AIGC 导演可行性 6 级标签

| 标签 | 含义 | 动作 |
|------|------|------|
| `DIRECT_GENERATABLE` | 单动作、少角色、空间清晰 | 可直接生成 |
| `ASSET_REQUIRED` | 身份/道具/场景必须先锁定 | 先建资产板 |
| `BOARD_REQUIRED` | 镜头/blocking/动作必须先画出来 | 先建故事板 |
| `POST_REQUIRED` | 精确文字/UI/VFX/音频更适合后期 | 加入后期职责 |
| `SPLIT_REQUIRED` | 内容超出 15 秒段落容量 | 拆分成更小段落 |
| `AVOID` | 模型不擅长或不稳定 | 重新设计节拍 |

### QA 10 重门禁

| 门禁 | 检查什么 |
|------|----------|
| 门禁1 | 故事：前提可理解、主人公有欲望、惊喜是挣来的 |
| 门禁1.5 | 来源：Source Packet 存在、不可改动项保留 |
| 门禁2 | AIGC 可行性：每个场景有标签、高风险已处理 |
| 门禁3 | 交接：角色/道具/风格有代码、段落有 in/out state |
| 门禁4 | 提示词：资产/故事板/Seedance 提示词分开、lint 通过 |
| 门禁5 | 生成检查：角色身份不错、干净帧无错误文字 |
| 门禁6 | 长剧/系列：角色状态追踪、资产数据库有版本 |
| 门禁7 | 合规/版权：品牌/肖像/版权/IP 已检查 |
| 门禁8 | 留存/平台：首帧钩子、平台气质匹配、AI 标注合规 |
| 门禁9 | 后期交付：字幕/精确文字/VFX/调色/导出规格有计划 |

## 工具选型速查

### 行业双雄

| 工具 | 适用场景 | 备注 |
|------|----------|------|
| **可灵 Kling** | 国内视频生成首选 | 国产绝对第一，亚洲审美适配 |
| **Runway Gen-4** | 国际视频生成首选 | 电影感质量高，生态成熟 |

### 按环节推荐

| 环节 | 推荐工具 |
|------|----------|
| 视频生成 | 可灵 / Runway / Pika / Luma |
| 图像生成 | Midjourney（分镜/关键帧行业标准）|
| 剪辑 | 剪映（国内）/ DaVinci Resolve（专业）|
| 配音 | ElevenLabs / 剪映 AI 配音 |
| 配乐 | Suno（4分钟完整歌曲，免费档够用）|
| 角色一致性 | ComfyUI + IP-Adapter + ControlNet |
| 质量增强 | Topaz Video AI 7.1（AI 超分到 8K）|

### 选型决策树

```
1. 预算多少？
   ├── ¥0 → 可灵免费档 + 剪映 + Suno 免费档
   ├── ¥500-3000/月 → Runway ($15/月) + 剪映 Pro + Suno Pro
   └── >¥3000/月 → Runway + Premiere + DaVinci Studio + ElevenLabs Pro

2. 目标观众在哪？
   ├── 中国/亚洲 → 可灵 + 即梦AI 优先
   └── 全球/欧美 → Runway + Pika + Luma

3. 技能水平？
   ├── 零基础 → 剪映 + 即梦AI + Melies
   ├── 有影视经验 → 可灵/Runway + 剪映→Premiere
   └── 技术流 → ComfyUI + SVD + AnimateDiff
```

## 行业表达 DNA

### 高频黑话

| 术语 | 含义 |
|------|------|
| 抽卡 | 同一 prompt 多次生成，从中挑选 |
| 出图率 | 生成素材中可用的比例（行业约 30-50%）|
| 一致性 | 跨镜头角色/风格/时序的连贯性 |
| 运镜 | 镜头运动方式（推/拉/摇/移/跟/环绕）|
| 炼丹 | LoRA 训练/模型微调 |
| 垫图 | 用参考图引导生成 |
| 跑片 | 批量生成素材 |
| 人均导演 | （调侃）AI 降低了创作门槛 |

### 外行破绽（10 个）

1. 把 Midjourney 当视频工具（MJ 只能生图）
2. 以为「写好一句 prompt 就能出片」
3. 问「AI 能不能自动拍一部电影」
4. 把 ChatGPT 生成的视频当成 AI 视频生成
5. 忽略音频环节——「只要画面好看就行」
6. 用「AI 导演」不知道这是圈内自嘲
7. 说「这个 AI 生成的」而不是「这个是可灵/Runway 出的」
8. 问「哪个模型最好」而不是「这个项目用什么模型组合」
9. 以为 AI 视频的「4K」和摄影机的「4K」是一回事
10. 第一次做就想「用 AI 拍一部电影」

## 质量基准（5 条可验证）

1. **跨镜头角色一致性**: ≥70% 观众能在无提示下识别同一个人（≥5个镜头）
2. **出片率 ≥40%**: 每10条素材≥4条可直接用于成片
3. **音频-画面同步误差 <200ms**: 口型对齐误差专业标准
4. **叙事密度 ≥1个情绪转折点/30秒**: 纯视觉堆砌完播率显著更低
5. **AI标注合规且不影响完播率**: 标注后完播率下降 <5%

## 与同类 skill 的差异

本 skill 定位为**创意导演知识框架**，强调审美判断和决策框架，而非工业生产链路。

| 维度 | 本 skill | 工业生产型 skill |
|------|----------|-----------------|
| 重点 | 创意导演的心智模型和质量判断 | 工厂化多 Agent 协作流程 |
| 输出 | 判断和决策框架 | 可执行的生产提示词文件 |
| 结构 | 7心智模型 + 10 Playbook + 5类型片 | 6子Agent + 流水线 + CSV台账 |
| 代码 | 无 | 5+ Python 脚本 |
| 适用 | 单人创意导演 | 团队/工厂化生产 |
