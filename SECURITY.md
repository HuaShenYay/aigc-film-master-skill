# 安全政策

## 支持版本

| 版本 | 支持状态 |
|------|----------|
| 1.0.x | ✅ 活跃维护 |
| < 1.0.0 | ❌ 已废弃 |

---

## 报告安全漏洞

如果你在本 skill 中发现安全相关问题（如敏感信息泄露、版权侵权内容链接等），请通过以下方式报告：

1. **GitHub Issue**：标记为 `🔒 Security`
2. **直接联系**：发 Message 给维护者

请**不要**在公开 Issue 中描述安全漏洞细节。

---

## 已知安全考量

本 skill 为纯知识库/框架文件，不涉及：
- ❌ 无代码执行
- ❌ 无文件写入操作
- ❌ 无外部 API 调用

**潜在风险**：

1. **外部链接**：references/ 目录下包含指向第三方网站的 URL。这些链接由贡献者维护，可能随时间变化。请以批判性眼光对待所有外部引用。
2. **版权信息**：工具价格、API 定价等信息可能过时。使用前请在官网核实最新价格。
3. **生成内容**：本 skill 提供的是知识框架，不对使用者的实际 AIGC 创作内容承担任何责任。

---

## 响应时间

- 安全报告：24h 内确认
- 修复计划：7 天内发布补丁版本
- 公开公告：随补丁一并发布
