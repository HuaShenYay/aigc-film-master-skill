## 技术术语

| 术语 | 英文 | 一句话解释 | 场景 |
|------|------|-------------|------|
| 扩散模型 | Diffusion Model | 通过逐步去噪生成图像的AI模型，是Stable Diffusion、Sora等的核心技术 | 文生视频、图生视频 |
| 潜在扩散 | Latent Diffusion | 在潜在空间（而非像素空间）进行扩散，大幅降低算力需求 | Stable Diffusion架构基础 |
| 视频扩散模型 | Video Diffusion Model | 将扩散模型扩展到时间维度，可生成连续视频帧 | Sora、Runway Gen-3 |
| Token | Token | AI模型处理文本的最小单位，也指标注训练数据的标签 | Prompt设计、模型训练 |
| 提示词 | Prompt | 用户输入给AI模型的文字指令，决定生成内容的方向和质量 | 所有AIGC生成场景 |
| 负面提示词 | Negative Prompt | 指定不希望出现的内容，过滤不良生成结果 | Stable Diffusion、ComfyUI |
| 种子 / 随机种子 | Seed | 控制AI生成随机性的数值，相同seed可复现相同结果 | 保持角色/场景一致性 |
| 微调 | Fine-tune | 在预训练大模型基础上，用少量特定数据继续训练以适应特定风格或任务 | 角色一致性、风格定制 |
| LoRA | LoRA (Low-Rank Adaptation) | 一种轻量级微调方法，通过低秩矩阵大幅减少可训练参数，快速适配特定风格/角色 | ComfyUI、WebUI 风格模型 |
| ControlNet | ControlNet | 在扩散模型中引入额外条件控制（深度图、边缘检测、姿态等）的神经网络结构 | 精确控制人物姿势、场景构图 |
| IP-Adapter | IP-Adapter | 将参考图像的特征注入生成过程，实现形象/风格保持 | 角色一致性、风格迁移 |
| 局部重绘 | Inpainting | 只重新生成图像/视频的指定区域，其余部分保持不变 | 修改角色表情、替换背景 |
| 扩图/外绘 | Outpainting | 向图像/视频原有边界外扩展内容 | 扩大画幅、补充场景 |
| 超分辨率/放大 | Upscaling / Super-resolution | 提升生成内容分辨率，使低清素材变高清 | 输出最终成片前的画质增强 |
| 帧间插值 | Frame Interpolation | 在现有帧之间插入新帧，提高视频帧率，使动作更流畅 | 24fps→60fps、慢动作效果 |
| 时序一致性 | Temporal Consistency | 确保视频各帧之间角色/场景/光照保持连贯，不产生闪烁或跳变 | 长视频生成的核心质量指标 |
| 运动笔刷 | Motion Brush | 在指定区域施加运动矢量，控制局部动态程度 | Runway Gen-3、Sora |
| 镜头控制 | Camera Control | 精确控制虚拟摄像机的运动（推拉摇移、环绕、俯仰等） | 电影感镜头设计 |
| 关键帧 | Keyframe | 指定视频中某些帧的特定内容，AI自动补全中间过渡 | 控制叙事节奏和画面变化 |
| ComfyUI | ComfyUI | 基于节点工作流的Stable Diffusion GUI，支持复杂管线搭建 | 专业AIGC视频制作流程 |
| Stable Video Diffusion | SVD | Stability AI发布的视频生成模型，可从单张图片生成短视频 | 图生视频、角色动画 |
| AnimateDiff | AnimateDiff | 将静态扩散模型扩展为动画生成器的训练框架 | 让文生图模型具备视频生成能力 |
| 文生视频 | Text-to-Video | 直接根据文字描述生成视频 | Sora、Pika、Gen-3 |
| 图生视频 | Image-to-Video | 以一张图片为起点，生成延续该画面的视频 | SVD、Runway |
| 视频转视频 | Video-to-Video | 将一段视频的风格、内容或形式进行转换 | 风格迁移、虚拟换景 |
| CFG Scale | Classifier-Free Guidance Scale | 控制生成结果与提示词贴合程度的参数，数值越高越贴合但可能过饱和 | 所有扩散模型生成 |
| Denoising Strength | 去噪强度 | 控制生成自由度，值越高AI"发挥"空间越大，值低则更忠实原图 | Img2img、视频转视频 |

## 创意/导演术语

| 术语 | 英文 | 一句话解释 | 场景 |
|------|------|-------------|------|
| AI 摄影 | AI Cinematography | 利用AI辅助完成摄影机运动设计、镜头模拟、光影预测等摄影工作 | 预可视化、虚拟拍摄 |
| AI 分镜/故事板 | AI Storyboarding | 用AI根据剧本自动生成分镜头脚本和画面预览 | 前期筹备、向客户提案 |
| 提示词工程（视频） | Prompt Engineering for Video | 为视频生成模型设计结构化提示词，控制叙事、风格、运镜等 | 所有文生视频工作流 |
| AI 驱动叙事 | AI-driven Narrative | 用AI分析观众偏好数据，辅助或自动生成故事线和剧情走向 | 互动剧、网文改编 |
| 生成式预可视化 | Generative Previsualization | 用AI快速生成拍摄前的预览画面，替代传统手绘或3D预演 | 降低前期成本、快速试错 |
| AI 调色 | AI Color Grading | 用AI自动完成色彩校正和风格化调色，或学习某导演/影片的色调风格进行迁移 | 后期调色、风格统一 |
| AI 视觉特效 | AI VFX | 用AI完成传统需要手工逐帧处理的视觉特效（抠像、物体移除、换天等） | 降低VFX成本 |
| 镜头构图（AI辅助） | Shot Composition (AI-assisted) | AI根据构图规则自动推荐或优化画面布局、主体位置、景深效果 | 拍摄现场辅助、后期重构图 |
| AI 剪辑工作流 | AI Editing Workflow | 用AI自动完成粗剪、场景分割、节奏匹配、音乐对齐等剪辑任务 | 缩短后期周期 |
| 多模态提示 | Multimodal Prompting | 同时输入文字+图像+音频+视频等多种模态作为生成条件 | 精确控制生成结果 |
| 虚拟演员/数字人 | Virtual Actor / Digital Human | 由AI生成或驱动的数字人物，可用于替代真人出演 | 降低演员成本、危险镜头 |
| AI 配乐/音效 | AI Music & Sound Design | 用AI根据画面情绪自动生成配乐和音效 | 低成本制作、快速出样片 |
| 风格迁移 | Style Transfer | 将一种视觉风格（如某导演、某画家的风格）迁移到视频帧上 | 统一MV、广告的视觉风格 |
| 一致性保持 | Consistency Enforcement | 跨镜头/跨场景保持角色形象、服装、场景细节不出现跳变的技术手段 | 长视频、系列短片制作 |
| 生成式延长/补全 | Generative Fill/Extend | AI自动补全视频片段的开头或结尾，或延长镜头持续时间 | 调整时长、补救穿帮镜头 |

## 行业黑话

| 术语 | 含义 | 使用场景 | 来源 |
|------|------|----------|------|

| 运镜 | 虚拟摄像机的运动方式（推拉摇移等），也指AI对镜头运动的控制能力 | 提示词设计、镜头语言 | 中文影视AIGC圈 |
| 抽卡 | 同一提示词多次生成、随机产出，从中挑选最满意的结果（类比抽卡游戏） | 文生图/视频社区 | 国内外AIGC社群 |

| 垫图 | 将参考图像作为输入条件引导生成方向 | Midjourney、Stable Diffusion | 图生图社区 |

| 魔改 | 对生成结果进行大幅度修改，或指用AI对原素材进行极端风格化处理 | 创意讨论 | 中文创作者社群 |
| Sora时刻 | 指某个AI视频生成工具达到"可骗过普通人眼"的质量拐点 | 行业分析文章 | 科技媒体/分析师 |

| 提示词工程师 | Prompt Engineer，专门从事提示词设计与优化的职业 | 招聘、自由职业 | 全球AIGC行业 |
| 幻觉 | Hallucination，AI生成了不符合物理规律或提示词要求的内容 | 技术讨论、质量评估 | AI学术界+工业界 |

## 行业标准

| 标准 | 说明 | URL |
|------|------|-----|
| **FVD**（Fréchet Video Distance） | 视频生成质量评估指标，衡量生成视频分布与真实视频分布之间的距离；数值越低质量越好；广泛用于学术研究和模型评测 | https://github.com/unibas-dmi/fvd-comparm |
| **FID**（Fréchet Inception Distance） | 图像生成质量评估指标，也可用于视频帧质量评估；衡量生成图像与真实图像在特征空间中的距离 | https://github.com/mseitzer/pytorch-fid |
| **CLIP Score** | 基于CLIP模型衡量生成图像/视频与提示词之间的语义匹配度；分数越高说明生成结果越符合提示词描述 | https://github.com/openai/CLIP |
| **IS**（Inception Score） | 早期图像生成质量评估指标，衡量生成图像的清晰度和多样性；现逐渐被FID取代 | — |
| **PSNR / SSIM** | 传统视频质量指标：PSNR（峰值信噪比）衡量失真程度；SSIM（结构相似性）衡量感知质量；常用于压缩/超分后的质量评估 | — |
| 《人工智能生成合成内容标识方法》（GB/T 强制性国标） | 中国网络安全技术强制性国家标准，与《标识办法》配套，详细规定显式/隐式标识的技术实现方式 | https://openstd.samr.gov.cn/ |
| ISO/IEC JTC 1/SC 42（人工智能标准化） | ISO与IEC联合成立的人工智能标准化分技术委员会，正在制定AI质量评估、风险管理等国际标准 | https://www.iso.org/committee/6794475.html |
| **EU AI Act 透明度义务** | 对生成式AI基础模型提供者设定义务：须公开用于训练的内容摘要、须确保生成内容以机器可读方式标记（水印） | https://artificialintelligenceact.eu/ |

## 信息不足标注

以下领域在现有搜索结果中信息有限，建议进一步调研：

1. **ISO/SMPTE 等影视行业专门标准**：关于AI工具在影视制作中的使用规范，目前尚未发现成熟的行业标准文档，可能仍在制定中。
2. **中国《著作权法》对AI生成内容的司法最新判例**：2024-2025年有多起相关案件（如"AI文生图著作权案"），但系统性的案例汇总仍有待补充。
