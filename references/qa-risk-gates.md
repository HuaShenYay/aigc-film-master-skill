# QA 与风险门禁

> 在判定一个包是"工业级"时使用，以及在每个主要交接之后使用。
> 默认后台运行门禁检查。不暴露门禁表格给用户，除非用户要求审计、审查、诊断或某个具体模块。

---

## 门禁 1：故事

- [ ] 前提能快速被理解
- [ ] 主人公有可见的欲望和压力
- [ ] 场景是因果的，不是装饰性的
- [ ] 结尾回溯开场或中心母题
- [ ] 惊喜是挣来的，不是随机的
- [ ] 情节解释之后仍有情感余味

---

## 门禁 1.5：来源/出处

- [ ] Source Packet 存在
- [ ] "不可改动"项已被保留
- [ ] 私人/客户资产有复用边界
- [ ] 外部参考材料被当作创作输入，不是复制表达

---

## 门禁 2：AIGC 可行性

- [ ] 每个场景/段落都有可行性标签
- [ ] 高风险效果已被拆分、简化或移至后期
- [ ] 没有段落携带过多命名角色、动作、文字职责或情绪转折
- [ ] 精确文字有后期/文字板计划
- [ ] 对白有时机和听众反应

---

## 门禁 3：交接

- [ ] 每个角色、场景、道具、风格、VFX、UI/文字元素都有代码或负责人
- [ ] 每个段落有入状态、主要动作、出状态、下游交接
- [ ] 下游系统可以直接生产提示词，无需猜测故事含义
- [ ] 导演选择已翻译成生产职责
- [ ] 品牌/留存/模型平台选择已翻译成生产职责

---

## 门禁 4：提示词生产

- [ ] 资产提示词和故事板提示词分开
- [ ] 故事板和 Seedance 提示词匹配镜头顺序和时间码
- [ ] Seedance 提示词在模型限制内
- [ ] 参考图有各自的明确职责
- [ ] 叙事帧禁止随机文字、字幕、水印、标签、损坏的 UI
- [ ] 最终提示词 Markdown 文件存在（当用户要求全流程时）
- [ ] 每个段落有对应的 Image2 故事板提示词和 Seedance 提示词
- [ ] 用户可以复制最终提示词，无需阅读内部 SOP 注释
- [ ] 提示词包不暴露私人路径、客户机密、不必要的来源细节
- [ ] 对话段落保留精确台词、VOICE_LOCK、台词节奏、停顿、听众反应表演
- [ ] 对话、对峙、追逐、交接、退出段落有 180 度轴线/屏幕方向锁定
- [ ] 群演场景有数量锁定、不同脸约束
- [ ] 每个 Seedance 段落有段落复杂度预算和一个主要戏剧功能
- [ ] 视觉风格参考已分离为风格文字锁定、安全风格图或资产/场景/道具参考

---

## 门禁 5：生成检查

**硬失败条件**（满足任一即失败）：

- [ ] 角色身份错误
- [ ]  recurring 场景错误
- [ ] 主人公道具错误
- [ ] 干净帧里出现不可读或损坏的文字
- [ ] 风格变成了不同媒介
- [ ] 情绪或动作不再讲述节拍
- [ ] 段落无法连接前/后段落

---

## 门禁 6：长剧/系列

- [ ] 每集/每段落后追踪角色状态
- [ ] 伏笔/回收表存在
- [ ] 资产数据库有版本
- [ ] 长流程生产存在角色状态表、场景状态表、道具状态表、镜头台账、take 评分库、可复用资产库
- [ ] 一次只送一个生产块下游
- [ ] 已生成的 takes 有评分并记录

---

## 门禁 7：合规/版权/IP

- [ ] 品牌/商标已检查
- [ ] 名人/肖像已获授权或有规避方案
- [ ] 私人图像/声音有使用许可
- [ ] 版权 IP 有法律评估
- [ ] 参考片风格引用在合理范围内
- [ ] 音乐/歌词有同步权或规避方案
- [ ] 声音克隆已授权
- [ ] 平台/比赛规则已核对
- [ ] 产品宣传声明有合规审查
- [ ] 内容安全风险已评估

---

## 门禁 8：留存/平台适配

- [ ] 首帧有"停滑"钩子
- [ ] 0-2 秒有信息压力
- [ ] 5 秒理解门槛已过
- [ ] 结尾记忆点存在
- [ ] 平台气质匹配（抖音/B站/YouTube/TikTok 各有不同）
- [ ] 画幅正确（9:16 / 16:9 / 1:1）
- [ ] AI 标注合规

---

## 门禁 9：后期交付

- [ ] 字幕/标题卡有后期计划
- [ ] 精确文字（合同/文件/招牌/手机 UI）有后期叠加方案
- [ ] 声音设计、音效、音乐有 cue sheet
- [ ] VFX 叠加有技术规格
- [ ] 调色方向已定义
- [ ] 导出规格匹配目标平台

---

## 门禁 10：版本/返修闭环

- [ ] 有版本日志
- [ ] 用户修改后有影响范围判断
- [ ] 已批准模块未被破坏
- [ ] 返修历史被记录
- [ ] 旧版本可追溯
